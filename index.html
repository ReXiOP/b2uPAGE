About the Project

**Bijoy to Unicode Converter** is a Python-based tool designed to convert text written in **Bijoy encoding** into **Unicode** format. This tool is particularly useful for those who deal with legacy text encoded using the Bijoy method, which is widely used in Bangladesh for typing Bengali text.

The tool is simple to use and does not require any complex setup. By just placing the files in the specified folder, users can easily convert multiple Bijoy-encoded `.docx` files into Unicode. This eliminates the need for manual conversion, saving time and improving productivity, especially when handling large batches of files.

The tool supports batch processing, which means you can convert multiple files in one go. It also features a real-time progress bar, providing feedback during the conversion process, so users know exactly where they are in the process.

Technologies Used

  • Python: The core language used to implement the conversion logic.
  • docx module: Used to handle the reading and writing of .docx files.
  • Bijoy Encoding Scheme: The encoding method that this tool works to convert into Unicode format.
  • Unicode Encoding: The universal character encoding standard that the tool outputs text in.

Why This Project?

In Bangladesh, the Bijoy encoding system has been historically popular for typing Bengali text. However, due to the global adoption of Unicode for text encoding, it’s often necessary to convert Bijoy-encoded files into Unicode. Manually performing this conversion can be time-consuming and error-prone, which is where this tool comes in. By automating the process, users can save time and ensure accuracy.

This tool is designed to be user-friendly and efficient, making it suitable for anyone needing to convert Bijoy-encoded documents—whether you're working with a single file or need to convert multiple files in bulk. It’s ideal for writers, researchers, educators, and developers working with Bengali text.

Potential Use Cases

  • Writers and Editors: Easily convert legacy documents written in Bijoy encoding to Unicode for modern usage.
  • Researchers: Convert old research papers or datasets that were written in Bijoy into a universally accessible format.
  • Educators: Use this tool to convert old textbooks or articles into Unicode format for digital learning platforms.
  • Software Developers: Developers working on Bengali text applications can use this tool to ensure compatibility with Unicode standards.

How to Use

Step 1: Prepare Your Files

  • Put your .docx files with Bijoy-encoded text into a folder (e.g., bijoy).
  • Create a folder for converted files (e.g., unicode).

Step 2: Run the Script

python app.py

This will convert all your files from the bijoy folder into Unicode and save them in the unicode folder.
